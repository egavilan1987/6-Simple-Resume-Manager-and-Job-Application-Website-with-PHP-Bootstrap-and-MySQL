Administrador de redes

DESCRIPTION

Nos encontramos en la búsqueda de una persona joven dinámica y proactiva, con estudios terminados o avanzados de Ingenieria que quiera desarrollarse profesionalmente. Con creatividad para la resolución de problemas que pueda desconocer, capacidad de trabajo en equipo y ganas de aprender. Se valorara: Conocimientos altos en Administración de sistemas, Linux Server, Windows server. Conocimientos medios en MySQL, PHP, JavaScript, Python Conocimeintos medios en networking, rewall, VPN. Conocimeinto Asterisk, IVR’s y VOIP . Tareas a desarrollar: Administrar/congurar sistemas basados en Linux para aplicaciones como, Asterisk, Firewalls, servers Webs, proxy, monitoreo. Administrar/congurar servidores basados en Windows Active Directory para el control de usuarios. Administrar/congurar infraestructura de Red, como switches/routers. Desarrollo y mantenimiento de scripts para la automatizacion de tareas como backups y alertas. Gestionar recursos de IT, insumos/repuestos, dar seguimiento a incidentes. Fomentar el trabajo en equipo con los demas integrantes del staff de IT. Procurar el correcto funcionamiento y proponer mejoras de los sistemas existentes. Flexibilidad horaria para guardias y tareas especiales fuera del horario establecido. Para postularse mencionar aspiraciones salariales !

REQUIREMENTS

Conocimientos altos en Administración de sistemas, Linux Server, Windows server. Conocimientos medios en MySQL, PHP, JavaScript, Python Conocimeintos medios en networking, rewall, VPN. Conocimeinto Asterisk, IVR’s y VOIP.
